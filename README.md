An intelligent irrigation system that uses **TinyML and ESP32** for real-time soil moisture monitoring and automated water management. The system performs machine learning inference directly on the ESP32, eliminating cloud dependency and enabling efficient operation in remote agricultural areas.

---

## 📌 Project Overview

Efficient water management is one of the major challenges in modern agriculture. Traditional irrigation methods often result in water wastage and reduced crop productivity.

This project presents a **TinyML-Based Smart Irrigation Decision System** that continuously monitors soil conditions and automatically controls irrigation based on the prediction generated by a lightweight machine learning model deployed on the ESP32.

## ⚙ Working Principle

1. Soil moisture sensor continuously measures soil conditions.
2. Sensor readings are sent to the ESP32.
3. TinyML Decision Tree model processes the input.
4. The model predicts whether irrigation is required.
5. Water pump is activated automatically if soil moisture is low.
6. LED and buzzer provide status indication.
7. Pump turns OFF when sufficient moisture level is achieved.

# 🧠 Machine Learning Model

The system employs a **Decision Tree-based TinyML model** deployed directly on the ESP32 microcontroller. By performing inference locally, the system achieves:

- Faster response time
- Reduced latency
- Lower power consumption
- No internet dependency
- Reliable operation in rural areas

The dataset contains soil moisture-related data used for training the machine learning model. It enables the system to classify soil conditions and determine when irrigation is necessary.
